# Prompt: Refactoring

**Objetivo:** Refatorar um trecho de código para melhorar legibilidade, manutenibilidade ou performance, sem alterar o comportamento externo.

**Instruções:**
Analise o código abaixo e proponha uma refatoração.

**Foco:**
1.  **Simplificação:** Reduzir complexidade ciclomática.
2.  **Nomes:** Melhorar nomes de variáveis e funções.
3.  **DRY (Don't Repeat Yourself):** Eliminar duplicações.
4.  **SOLID:** Aplicar princípios SOLID onde fizer sentido.
5.  **Modernização:** Usar features mais novas da linguagem (se aplicável).

**Saída:**
- **Código Refatorado:** O código completo após as mudanças.
- **Explicação:** Por que as mudanças foram feitas e quais os benefícios.
