# Prompt: Pull Request Description

**Objetivo:** Criar uma descrição detalhada e profissional para um Pull Request (PR).

**Instruções:**
Com base no título do PR e nas alterações realizadas, gere uma descrição completa para facilitar a revisão.

**Estrutura da Descrição:**
1.  **Resumo:** O que este PR faz?
2.  **Tipo de Mudança:** (Marque com x)
    - [ ] Bug fix
    - [ ] Nova feature
    - [ ] Breaking change
    - [ ] Documentação
3.  **Detalhes da Implementação:** Explique as principais decisões técnicas.
4.  **Como Testar:** Passos para validar as mudanças.
5.  **Screenshots/Logs:** (Se aplicável)
6.  **Checklist:**
    - [ ] Testes passaram
    - [ ] Documentação atualizada
    - [ ] Linting verificado
